# Pemrograman Berorientasi Objek — Tugas Method Java

## Deskripsi
Project ini merupakan implementasi konsep Method dalam Java sebagai bagian dari tugas mata kuliah Pemrograman Berorientasi Objek. Program dibuat dengan memanfaatkan method untuk memisahkan proses perhitungan dari program utama (main). Setiap perhitungan dibuat dalam method tersendiri agar kode lebih terstruktur dan juga mudah dipahami.

---

## Konsep yang Digunakan
- Method (fungsi dan prosedur)
- Pemanggilan method
- Perulangan for
- Percabangan if
- Operator aritmatika
- Operator modulus (%)
- Perhitungan faktorial
- Input menggunakan Scanner
